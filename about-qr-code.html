About the QR code


  • The QR code on front page is meant for providing additional information relevant to the book subject.
  • When the code is scanned with smartphone camera, the link buttons on different topics will be visible.
  • On clicking desired button, the additional information in PDF, JPEG or MP4 format will open to read/ view.
  • Significance of this concept-
    • This is an effort to cross the limits of book to elaborate the topic with more depth through infotainment.
    • Technology is changing fast, so some very recent information can be in the hands of reader through the QR code.
    • The ultimate aim is to make the text book interactive.
